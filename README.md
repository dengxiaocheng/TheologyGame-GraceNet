# 恩典之网 / Net of Grace

基于拉纳神学的连接解谜游戏。

**在线试玩**: [https://dengxiaocheng.github.io/TheologyGame-GraceNet/](https://dengxiaocheng.github.io/TheologyGame-GraceNet/)

## 技术栈
- 纯前端 HTML/CSS/JavaScript
- Canvas 渲染
- 无需构建工具，直接浏览器打开即可

## 开发
```bash
# 本地运行
python3 -m http.server 8000
# 访问 http://localhost:8000
```
